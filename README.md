Numba HSA Examples
==================

This repository contains several example of HSA GPU programming using numba.


Run Tests
---------

Run full test suite:

```bash
./runtests.sh
```

Selectively run tests from subdirectories:

```bash
./runtests.sh <subdirectory>
```

Example:

```
./runtests.sh numba_hsa_examples/kerneldensity
```
